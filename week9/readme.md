# 9주차 과제
## 자바 SWING을 이용한 피자 주문서 작성
### 1. 제시된 코드를 참고하여 기본 UI 구현
제시된 코드를 그대로 따라하면 피자주문을 제목으로 하는 창이 표시되고 종류와 사이즈는 라디오 버튼으로 박스로 묶여 있다.  
먼저 JFrame의 제목에 학번과 이름을 포함해야한다.  
나머지 패널은 그대로 두고 중앙에는 주문수량입력을 위한 패널을 추가하고 종류와 사이즈 패널은 라디오 버튼에서 레이블과 콤보박스로 수정한다.  
모든 패널이 들어간 뒤 적절히 표시될 수 있도록 JFrame 크기를 조절한다.  
### 2. JFrame의 setSize 메소드로 화면 크기 조정 (25점)
```java
    setSize(500, 250);
```
각 컴포넌트가 모두 표시될 수 있도록 적절하게 조절하면 된다.
### 3. JFrame의 setTitle 메소드로 학번 이름을 제목에 포함 (25점)
```java
    setTitle("피자주문 학번 이름");
```
자신의 학번과 이름을 제목에 포함하면 된다.
### 4. 주문수량입력을 위한 패널 추가 (25점)
주문 수량을 입력하기 위한 패널을 추가한다. JLabel과 JTextField를 이용하면 쉽게 구현할 수 있다.  
중간에 이미 토핑 패널이 있는데 어떻게 패널을 추가할 수 있는지 고민한 학생이 몇몇 있는 것 같다. 
패널을 하나 더 만들고 패널에 토핑 패널과 주문 수량 입력 패널을 추가하면 두 패널을 중간에 배치할 수 있다.
### 5. SizePanel의 컴포넌트를 레이블과 콤보상자로 변경 (25점)
레이블과 콤보상자 추가를 빼먹고 기존 코드 그래도 제출한 학생이 있다.  
라벨을 JLabel이 아닌 awt의 Label을 사용한 경우가 있었는데 Label은 유니코드를 지원하지 않아 한글에 제대로 표시될 수 없는 문제가 발생한다.

<br>
대부분의 학생들이 피자 주문서를 잘 만들어서 제출하였다. 실습 시간에 사용했던 코드를 제공하여 쉽게 수행할 수 있었던 과제가 아니였나 생각한다.
